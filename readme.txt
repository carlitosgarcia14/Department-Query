READ ME

Take the following steps to run this program from cmd:

1. Download the required files and save them to your desktop.
2. Open cmd or terminal.
3. Enter the command: “cd Desktop” and press enter.
4. Enter the command: “javac DepartmentQuery.java” and press enter.
5. Enter the command: “java DepartmentQuery” and press enter.
6. Once the program begins to run enter “employee-sample.csv” as user input to query the file.

The program should’ve now ran successfully. 

This program requires the following files to run successfully:
DepartmentQuery.java
employee-sample.csv
